# Design Notes

## Current Direction

- Modern, minimal desktop todo UI
- Purple accent color for primary actions and completion states
- Subtle lavender-to-cool-light linear gradient background
- Soft shadows, rounded corners, and generous spacing

## Layout

- Centered card container
- Title at top left
- Single text input with adjacent Add button
- Simple vertical task list with checkbox and delete action
